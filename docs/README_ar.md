# مشروع WRO-G15 (بالعربية)

هذا التوثيق موجه للطلاب المبتدئين لفهم مشروعنا خطوة بخطوة.

## المكونات
- ESP32 DevKit V1
- L298N Driver
- محركات DC
- Servo SG90
- حساس Ultrasonic
- حساس ألوان TCS3200
- بطاريات 18650

## كيف يشتغل الروبوت؟
1. نشغل البطارية.
2. نضغط زر START.
3. الروبوت يمشي للأمام.
4. إذا في عائق → يلف.
5. يقرأ ألوان الزوايا (أزرق/برتقالي).
6. بعد 3 لفات يتوقف.

## ماذا تعلمنا؟
- توصيل مكونات الروبوت.
- برمجة ESP32 بلغة C++.
- تقسيم الكود إلى ملفات منظمة.
